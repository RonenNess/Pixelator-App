Update Pixelator

Instructions

To update Pixelator, please follow these instructions:

  • Download the latest installer from the Download Page.
  • Make a backup of your LICENSE file outside the Pixelator folder.
  • Uninstall Pixelator (by deleting its folder or using the installer if you have one).
  • Install Pixelator again from the new installer and copy the LICENSE file back inside.

Note: a proper update mechanism will be added in later versions. Meanwhile updates have to go manually.
