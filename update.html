Update Pixelator

Instructions

To update Pixelator, please follow these instructions:

  • Download the latest installer from the Download Page.
  • Super important: make a backup of your LICENSE file outside the Pixelator folder.
  • Uninstall Pixelator (deleting its folder is enough).
  • Install Pixelator again from the new installer and copy the license back inside.

Note: a proper update mechanism will be added in later versions. Meanwhile updates have to go manually.
