Pixelator 1.0.0

Release Date: 10/02/2018.

  • First stable release.

Pixelator 1.0.1

Release Date: 20/02/2018.

  • Fixed bug in languages that use comma instead of dot for floating point.
  • Fixed bug that caused small pictures to look blurry in preview panel.
  • Fixed some UI typos.
  • Better error handling for non-critical problems.
  • Added scrollbar for smaller resolutions.
  • Building preview now happens in background, UI is not blocked by it.
  • Added progressbar to indicate preview generating progress.
  • Added manual update mode, where you must click to update preview.
  • Added number input for all sliders, so you can type numbers directly instead of using the slide bar.
  • Added support in dragging files into the app.
  • The app will now check for updates automatically when starting up.

Pixelator 1.0.2

Release Date: 05/03/2018.

  • Remade entire UI, now compatible with Linux.
  • Fixed bug with exit code.
  • Added codecs module to build.
  • Added debug license commands.
  • Added auto-update of source image.
  • Increased max colors to 256 in app.
  • Fixed bug with paths containing spaces.
  • More flexible preview panels.
  • New project file format.

Pixelator 1.0.3

Release Date: 07/03/2018.

  • Added Mac OS support.
  • Added apply-license option from UI.

Pixelator 1.0.4

Release Date: 30/06/2018.

  • Turned 'Pixelate' factor to float, allowing finer tuning.
  • Added option to load color palette from file.
  • Added built-in color palettes.
  • Option to make stroke color based on palette instead of default black.

Pixelator 1.0.5

Release Date: 06/10/2018.

  • Changed default output file format to png.
  • Added option to copy preview to clipboard.
  • Added option to save / reset default settings.
  • Added 'Undo'.

Pixelator 1.0.6

Release Date: 12/12/2018.

  • Added option to set Pixelation factor to 1, ie keeping original resolution.
  • Added option to use color palette from file with or without dither (previously it was only with dither).
  • Fixed scrollbar to be for options only and not whole screen (so preview don't move up when you scroll).
  • Improved error messages in app to provide better support.
  • Added outline color selection.
  • Divided UI into categories, made them collapsable.
  • Made a proper preview image options (collapsable).
  • Added option to show preview images in their actual size.
  • Fixed default color selection value (transparent).
