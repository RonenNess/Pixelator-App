Pixelator 1.0.0

Release Date: 10/02/2018.

  • First stable release.

Pixelator 1.0.1

Release Date: 20/02/2018.

  • Fixed bug in languages that use comma instead of dot for floating point.
  • Fixed bug that caused small pictures to look blurry in preview panel.
  • Fixed some UI typos.
  • Better error handling for non-critical problems.
  • Added scrollbar for smaller resolutions.
  • Building preview now happens in background, UI is not blocked by it.
  • Added progressbar to indicate preview generating progress.
  • Added manual update mode, where you must click to update preview.
  • Added number input for all sliders, so you can type numbers directly instead of using the slide bar.
  • Added support in dragging files into the app.
  • The app will now check for updates automatically when starting up.
